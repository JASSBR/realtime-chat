# Real-time Chat Application

## Project Goal
WebSocket chat app with rooms, typing indicators, online users, and message history.

## Key Rules
- Write clean, production-quality code with no placeholder/TODO comments
- Every feature must be fully functional — no stubs
- Use modern JavaScript (ES2020+), semantic HTML5, modern CSS
- Zero external dependencies unless absolutely necessary (use native Node.js / browser APIs)
- Include proper error handling and edge cases
- Code must run immediately after implementation — test everything
- Write a comprehensive README.md with screenshots section, features list, and usage instructions

## Tech Constraints
- Category: fullstack
- Difficulty: intermediate
