# Real-time Chat Application

WebSocket chat app with rooms, typing indicators, online users, and message history.

## Tech Stack

- React
- Node.js
- MongoDB
- Express

## Getting Started

```bash
# Clone the repo
git clone https://github.com/JASSBR/realtime-chat.git
cd realtime-chat

# Install dependencies (if any)
npm install

# Run the project
npm start
```
